# Pintos

> -   OS프로젝트는 PintOS의 코드를 직접 수정해가며 진행하는 프로젝트입니다.
> -   PintOS는 2004년 스탠포드에서 만들어진 교육용 운영체제예요. 우리 프로젝트는 이를 기반으로 KAIST 권영진 교수님 주도 하에 만들어진 `KAIST PintOS`로 진행됩니다.

핀토스 프로젝트의 목표는 운영체제를 만들어보며 운영체제의 이해도를 높이는 것이다.
